Linguagens

Algumas linguagens que conheço

  • JAVA

    Java é uma linguagem de programação orientada a objetos, conhecida por sua portabilidade e robustez, amplamente utilizada para desenvolver aplicações web, móveis e empresariais.

  • HTML

    Linguagem padrão usada para criar e estruturar páginas na web, permitindo a formatação de texto, imagens e outros elementos multimídia em um documento visualmente agradável e navegável.

  • CSS

    Linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML, permitindo controlar o layout, fontes e outros aspectos visuais das páginas de forma eficiente e flexível.

  • JAVASCRIPT

    Linguagem de programação, amplamente utilizada para criar páginas web interativas, permitindo a manipulação de conteúdo, animações e comunicação com servidores.

  • PYTHON

    Linguagem de programação conhecida por sua simplicidade e legibilidade, amplamente utilizada para desenvolvimento web, análise de dados, automação e muito mais.

  • MYSQL

    Sistema de gerenciamento de banco de dados, utilizado para armazenar, organizar e recuperar dados em aplicações web e empresariais, conhecido por sua eficiência, escalabilidade e facilidade de uso.
